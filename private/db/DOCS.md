
# SQL [folder](./sql/)
Use of SQL folder is to have it as a reference point of how the data conforms to (tables, variables, attributes) in addition an automatic database intialization with the help of [init.js](./init.js) that provides a clear database initialization of the project.

# Init.js
Simply initializes the existing sql files inside the sql folder into the database to ensure that the project will run by simply writing `node .`.

<br><br><br>
Further idea of this directory is to utilize database functions, however they have been implemented directly in functions of need of database functions.
